FILTRO DE BUSCA NA ABA EMPRESAS

Na página Admin → Empresas, criar sistema de busca para localizar empresas rapidamente.

Adicionar campo de pesquisa no topo da tabela.

Campo:

Pesquisar empresa

A busca deve funcionar por:

Nome da empresa
Email
Contato

Implementar filtro dinâmico.

Exemplo:

Digitando:

Netflix

Sistema retorna apenas empresas que contenham esse nome.

Adicionar também filtros adicionais:

Empresa ativa / inativa
Tipo de cliente

Tipos:

Cliente semanal
Cliente mensal
Cliente pontual

Se não houver resultados mostrar mensagem:

Nenhuma empresa encontrada.

Garantir que a busca funcione mesmo com grande quantidade de empresas.

Tipo de cliente

Opções:

Cliente semanal
Cliente mensal
Cliente pontual

Comportamento:

Cliente semanal
Recebe lembrete semanal de pedidos.

Cliente mensal
Permite programação de pedidos mensais.

Cliente pontual
Não recebe emails automáticos.

JANELA GLOBAL DE PEDIDOS

Criar sistema de controle de janelas de pedidos.

Regra padrão:

Pedidos liberados até quinta-feira às 12:00.

Após esse horário pedidos devem ser bloqueados.

Mensagem exibida ao cliente:

Prazo de pedidos encerrado.

CONTROLE GLOBAL

Criar no admin:

Controle Global de Pedidos

Com duas opções:

Abrir pedidos
Fechar pedidos

Se estiver fechado:

Nenhum cliente pode fazer pedidos.

Mensagem:

Pedidos temporariamente indisponíveis.
