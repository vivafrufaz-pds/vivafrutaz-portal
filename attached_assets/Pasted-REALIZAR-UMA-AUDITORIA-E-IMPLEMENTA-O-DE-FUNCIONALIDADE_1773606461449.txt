2) OPÇÃO DE FREQUÊNCIA DE PEDIDOS PARA CLIENTE

Na área do cliente → Aba Pedidos

Adicionar campo de seleção:

"Tipo de pedido preferido"

Opções:

Pedido semanal
Pedido mensal
Pedido pontual

Se cliente escolher:

Pedido semanal → sistema abre janela de pedido toda semana.

Pedido mensal → cliente faz um pedido válido para todo o mês.

Pedido pontual → cliente compra apenas quando quiser.

Essa preferência deve ser salva no cadastro do cliente.

Melhorar comportamento.

Safra encerrada
Produto indisponível

verificar pedidos existentes que contenham esse produto.

"Produto X saiu de safra. Existem pedidos ativos contendo esse produto."

empresa
pedido

"Gerenciar substituição"

Opções:

substituir produto por outro

Essas alterações devem atualizar automaticamente:

pedido
