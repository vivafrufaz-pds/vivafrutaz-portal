IMPORTANTE

NÃO criar novas abas.
NÃO alterar a navegação do sistema.
NÃO mudar a estrutura das páginas existentes.

A melhoria deve acontecer somente dentro da área atual:

Empresas
→ Editar empresa
→ Escopo Contratual

Manter exatamente o layout atual da tela.

--------------------------------------------------

MELHORIA DENTRO DO ESCOPO CONTRATUAL

Atualmente o escopo possui campos:

Dia da semana
Categoria
Produto
Quantidade

Quero adicionar novos campos diretamente nessa mesma área.

Novos campos por item:

Preço unitário
Total do item
Custo médio (opcional)

--------------------------------------------------

REGRAS DE FUNCIONAMENTO

Cada item do escopo deve funcionar como um item de pedido ou item de nota fiscal.

Estrutura final do item:

Dia da semana
Categoria
Produto
Quantidade
Preço unitário (editável)
Total do item (automático)

Regra:

Total do item = Quantidade × Preço unitário

O campo total não deve ser editável manualmente.

--------------------------------------------------

CÁLCULO DE CUSTO MÉDIO AUTOMÁTICO

Na parte inferior da lista de itens do escopo, mostrar um pequeno painel de resumo com:

Quantidade total
Valor total
Custo médio unitário

Regras de cálculo:

Valor total = soma de todos os totais dos itens

Quantidade total = soma de todas as quantidades

Custo médio = Valor total ÷ Quantidade total

Esse cálculo deve atualizar automaticamente conforme os itens forem editados.

--------------------------------------------------

FLAGS DE CONFIGURAÇÃO

Adicionar duas flags de controle dentro do escopo contratual.

Flag 1:

"Calcular custo médio automaticamente"

Se ativada:
o sistema calcula o custo médio automaticamente.

Se desativada:
o usuário pode inserir o custo médio manualmente.

--------------------------------------------------

Flag 2:

"Preço automático dos produtos"

Se ativada:
o sistema pode preencher preços automaticamente a partir do cadastro de produtos.

Se desativada:
o usuário deve inserir o preço manualmente.

--------------------------------------------------

IMPORTANTE

Essas funcionalidades devem estar disponíveis apenas para empresas classificadas como:

Cliente Contratual

Para empresas:

Pontual
Semanal
Mensal

essas funções devem permanecer bloqueadas.

--------------------------------------------------

AUDITORIA DO FLUXO DO SISTEMA

Após implementar essas melhorias, execute uma auditoria no sistema para verificar se o fluxo de pedidos e faturamento está funcionando corretamente para clientes contratuais.

Analise o seguinte fluxo:

Escopo contratual
↓
Geração de pedidos
↓
Pedidos aparecem na aba "Pedidos"
↓
Financeiro visualiza pedidos
↓
Financeiro gera faturamento
↓
Sistema gera Nota Fiscal
↓
Sistema gera DANFE

Verificar especificamente:

1. Onde os pedidos gerados a partir do escopo contratual aparecem.
2. Se esses pedidos aparecem corretamente na aba de pedidos do sistema.
3. Se o financeiro consegue visualizar esses pedidos.
4. Se o financeiro consegue gerar a nota fiscal para esses pedidos.
5. Se os dados do escopo (produto, quantidade, preço) estão sendo utilizados corretamente na nota.
6. Se o DANFE é gerado corretamente.

--------------------------------------------------

RELATÓRIO FINAL

Após a auditoria, apresentar um relatório contendo:

Pontos que estão funcionando corretamente.
Possíveis erros no fluxo.
Funcionalidades faltando.
Sugestões de melhoria para garantir que o fluxo de pedido → faturamento → nota fiscal funcione sem erro.
