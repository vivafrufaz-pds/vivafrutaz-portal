SUPER PROMPT – CORREÇÃO GLOBAL + OCORRÊNCIAS + PAINEL DE ACOMPANHAMENTO VIVAFRUTAZ

Executar atualização estrutural completa no sistema VivaFrutaz com objetivo de:
	•	corrigir erro de upload de imagens
	•	eliminar erro “request entity too large”
	•	eliminar erro “objeto não encontrado”
	•	estabilizar renderização do frontend
	•	criar sistema de acompanhamento de ocorrências para clientes

Todas as correções devem ser aplicadas automaticamente para:
	•	administradores
	•	equipe interna
	•	clientes existentes
	•	novos clientes criados futuramente

Nenhuma atualização deve funcionar apenas para alguns usuários.

⸻

CORREÇÃO DO ERRO “REQUEST ENTITY TOO LARGE”

Aumentar limite de upload do servidor.

Se o backend utilizar Express configurar:

bodyParser.json({ limit: “25mb” })

bodyParser.urlencoded({
extended: true,
limit: “25mb”
})

Se o sistema utilizar multer configurar:

limite por arquivo: 5MB
máximo de arquivos por ocorrência: 5

⸻

VALIDAÇÃO DE TAMANHO DE IMAGEM NO FRONTEND

Antes de enviar imagem validar tamanho.

Se arquivo ultrapassar:

5MB

bloquear envio.

Mostrar mensagem:

“Cada imagem deve ter no máximo 5MB.”

⸻

OTIMIZAÇÃO AUTOMÁTICA DE IMAGEM

Antes do upload aplicar compressão automática.

Redimensionar imagens muito grandes.

Limite recomendado:

1600px largura máxima.

⸻

UPLOAD DE MÚLTIPLAS IMAGENS

Permitir até:

5 imagens por ocorrência.

Formatos aceitos:

JPG
JPEG
PNG

Mostrar contador:

1/5
2/5
3/5

⸻

REMOÇÃO DE IMAGEM

Cada imagem anexada deve exibir:

ícone de lixeira.

Ao clicar na lixeira:

remover imagem da lista
remover imagem do envio do formulário

⸻

CORREÇÃO DO ENVIO DA OCORRÊNCIA

Garantir envio utilizando:

FormData

Campos enviados:

tipo de ocorrência
descrição
telefone
email
empresa
imagens anexadas

Corrigir erro que impede salvar ocorrência no banco.

Criar área onde o cliente possa acompanhar suas ocorrências.

“Minhas Ocorrências”

Informações exibidas:

número da ocorrência
data de abertura
tipo da ocorrência
status
última atualização

imagens anexadas
data de abertura
status atual

Criar campo interno para administradores responderem ocorrência.

Administradores podem:

enviar mensagem
