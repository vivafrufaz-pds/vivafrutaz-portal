TIPOS DE CLIENTE NO CADASTRO DE EMPRESAS

Na página:

Admin → Cadastro de Empresas

Ajustar campo:

Tipo de cliente

Adicionar nova opção:

Cliente semanal

Lista final deve conter:

Cliente semanal
Cliente mensal
Cliente pontual

Comportamento:

Cliente semanal
Faz pedidos todas as semanas.

Cliente mensal
Pode ter programação mensal de pedidos.

Cliente pontual
Cliente esporádico que não recebe notificações automáticas.

MÓDULO 2

TAXA ADMINISTRATIVA AUTOMÁTICA

Criar sistema de aplicação automática de taxa administrativa por empresa.

No cadastro de empresas adicionar campo:

Taxa administrativa

Criar botões de seleção rápida.

Botões disponíveis:

GRSA → 27%
SODEXO → 18%

Esses botões devem automaticamente preencher o campo de taxa administrativa.

Exemplo:

Selecionar GRSA

Sistema define automaticamente:

Taxa administrativa = 27%

Selecionar SODEXO

Sistema define automaticamente:

Taxa administrativa = 18%

O administrador também pode inserir valor manual se necessário.

IMPORTANTE:

Clientes não podem visualizar a taxa administrativa.

Somente administradores do sistema podem ver essa informação.

O sistema deve calcular automaticamente o preço final exibido ao cliente.

Exemplo:

Produto base = 2,00
Taxa administrativa = 27%

Preço final mostrado ao cliente:

2,54

MÓDULO 3

FILTRO DE PRODUTOS NA TELA DE PEDIDOS DO CLIENTE

Na página onde o cliente realiza pedidos adicionar filtros.

Filtros disponíveis:

Filtrar por data de entrega.

Exemplo:

Segunda-feira
Quarta-feira
Sexta-feira

Quando o cliente selecionar segunda-feira, o sistema deve permitir pedidos apenas para essa data.

Não permitir selecionar múltiplos dias no mesmo pedido.

Exemplo:

Se o cliente estiver fazendo pedido de segunda-feira, bloquear seleção de quarta-feira.

⸻

MÓDULO 7

RECUPERAÇÃO DE SENHA

Remover erro técnico exibido quando senha está incorreta.

Mensagem atual:

Invalid credentials or inactive account

Substituir por mensagem amigável:

Email ou senha incorretos.

Adicionar botão:

Esqueci minha senha

Fluxo de recuperação:

Cliente informa email.

Sistema envia solicitação de recuperação de senha.

Essa solicitação deve aparecer no painel administrativo.

Admin pode:

Aprovar redefinição de senha
Gerar nova senha
Enviar nova senha ao cliente
