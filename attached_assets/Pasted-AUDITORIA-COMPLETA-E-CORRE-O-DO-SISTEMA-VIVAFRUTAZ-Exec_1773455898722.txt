AJUSTE NO CADASTRO DE EMPRESAS

Na tela:

Empresas → Nova Empresa

Atualmente existe campo:

Email (Login)

Alterar nome do campo para:

LOGIN

Esse campo será utilizado exclusivamente para acesso ao portal do cliente.

--------------------------------------------------

E-mail de notificações

Esse e-mail será utilizado para envio automático de:

notificação de pedidos
cotações
alertas
confirmações
ocorrências registradas

--------------------------------------------------

TRATAMENTO DE ERROS EM OCORRÊNCIAS

Se ocorrer falha no envio:

não quebrar a página.

Mostrar mensagem:

"Não foi possível registrar a ocorrência. Tente novamente."

Erro técnico deve ser registrado apenas no log interno.

--------------------------------------------------

CORREÇÃO DO ERRO EM PEDIDOS PONTUAIS

Executar auditoria no módulo:

Pedidos Pontuais

Verificar:

validação do formulário
dados enviados para API
estrutura da requisição
salvamento no banco
retorno da API

--------------------------------------------------

FLUXO CORRETO DO PEDIDO PONTUAL

Registro criado no log do sistema

VALIDAÇÃO DE DADOS DO PEDIDO
